About the Dragonfly GraphQL API

The API is found under route /api

Try to query it with 

{
    specie(items_id:"f0264cd93fc96310d176cbde4c32be52") {
        scientific_name
        description
        local_names
        behaviour
        habitat
        similar_species
        distribution
        flight_period
        size {
            length
            wingspan
        }
    }
}
or 
{
    species
}
